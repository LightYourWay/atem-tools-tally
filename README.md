# atem-tools-tally

A quick example of the atem-tools npm module configured for use as tally light controller.

## install
```bash
git clone https://github.com/LightYourWay/atem-tools-tally.git && \
cd atem-tools-tally && \
npm install
```

## usage
Modify `main.ts` to your needs. Then run:
```bash
npm start
```
